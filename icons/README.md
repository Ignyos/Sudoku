- icon-maskable-192.png (192×192)
- icon-maskable-512.png (512×512)

## Design Guidelines:
- Use a simple, recognizable Sudoku grid design
- Primary color: #2563eb (blue)
- Background: #0f172a (dark blue) or transparent
- Ensure the design works at small sizes
- For maskable icons, keep important content in the "safe zone" (80% of canvas)

## Tools to Generate Icons:
- https://realfavicongenerator.net/
- https://www.pwabuilder.com/imageGenerator
- https://maskable.app/ (for maskable icons)
